== Description ==

Quickly send notification to Push enabled devices from WordPress dashboard. The Plugin also facilitates customized JSON data support for your applications.  Currently supporting only Parse.com Push Service (for iOS, Android, Windows etc.). One must have cURL enabled on the host and some patience.

== Installation ==

1. Upload 'wp-quick-push' folder to the '/wp-content/plugins/' directory.
2. Activate the WP Quick Push Plugin through the 'Plugins' menu in WordPress.
3. Configure 'Application Name', 'Application ID', 'REST API Key' values from Parse.com application in Plugin's Settings Page.

== Frequently asked questions ==

= How to get Parse.com Application Keys? =
Visit http://parse.com/ and Login|Signup for an account. Create a new Push|Choose Existing application and navigate to Settings tab on top. Under settings tab, click 'Keys' from the left pane, to list all the API information for your Push Application.
